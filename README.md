# Health-monitoring-system
This project is a Smart Elderly Health Monitoring Dashboard that displays real-time health parameters such as heart rate, SpO₂, body temperature, and steps using actual sensor data from an ESP device. The sensor data is transmitted to Firebase and visualized on a web dashboard with live charts and status indicators. The system includes an alert mechanism that detects abnormal conditions and notifies users, along with an option to resolve alerts. The dashboard is responsive, visually appealing, and user-friendly, making it suitable for real-time remote health monitoring of elderly patients.
